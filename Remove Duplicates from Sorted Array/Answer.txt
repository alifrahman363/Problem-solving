
Explanation: 
if the array is empty return 0 
if not then take a variable 'idx' which holds the value 1 
compare the index 1 and 0 of the 'nums' array
if they dont match then 
	index 0 stays the same 
	index 1 changes with index 1 
	index value increases

again compare index 2 and 1 of the 'nums' array
if the dont match then 
	index 1 stays the same 
	index 2 changes with the value of index 2 
	index value increases
 


class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        idx = 1
        if len(nums) == 0:
            return 0 
        
        for i in range(1,len(nums)): 
            if (nums[i] != nums[i-1]):
                nums[idx] = nums [i]
                idx +=1 
        return idx
